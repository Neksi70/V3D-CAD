Volme3D Druck-Helfer - Einrichtung (Windows, EINE Datei, ohne Installation)
===========================================================================

Wozu?  Damit die STL-Bibliothek in Volme3D
  - deine heruntergeladenen STLs aus dem Downloads-Ordner einsammeln kann
    (Chrome verbietet Web-Apps den direkten Zugriff auf Downloads), und
  - "Drucken" eine Datei mit EINEM Klick in Bambu Studio / Orca Slicer oeffnet.

Der Helfer laeuft im Hintergrund und lauscht nur lokal (127.0.0.1) - NICHT aus
dem Internet erreichbar, reagiert nur auf deine Volme3D-Adresse. Keine
Installation noetig (PowerShell ist auf Windows schon da).

AUTOMATIK: Sobald der Helfer laeuft, verschiebt er neue STL/3MF/OBJ aus dem
Downloads-Ordner alle ~30 Sekunden von selbst in die Bibliothek
(Dokumente\Volme3D-STL) - Downloads wird dabei aufgeraeumt. Die Galerie in
Volme3D aktualisiert sich automatisch. Du musst nichts mehr anklicken.

Einrichten (so einfach wie moeglich)
------------------------------------
  1) In der STL-Bibliothek auf  "Druck-Helfer.bat herunterladen"  klicken.
     (Falls Browser warnt: "Beibehalten" / "Trotzdem herunterladen".)
  2) Die Datei  Volme3D-Druck-Helfer.bat  doppelklicken.

  Es erscheint ein Fenster:
    "Volme3D Druck-Helfer laeuft auf http://127.0.0.1:7777"
    "Gefundene Slicer: bambu -> ... , orca -> ..."
    "Autostart eingerichtet."

  -> Fenster offen lassen, zurueck zu Volme3D, STL-Bibliothek neu oeffnen:
     der Status-Punkt wird gruen. Fertig.

Das ist nur EINE Datei - PowerShell steckt darin. Nichts weiter noetig.

Autostart
---------
  Beim ersten Start traegt sich der Helfer automatisch (unsichtbar) in den
  Windows-Autostart ein. Ab dem naechsten Anmelden laeuft er von selbst - du
  musst die Datei nicht mehr doppelklicken.

Falls "Windows hat Ihren PC geschuetzt" erscheint
-------------------------------------------------
  Auf "Weitere Informationen" -> "Trotzdem ausfuehren" klicken. (Die Datei ist
  ein einfaches Skript, das nur lokal arbeitet.)

Slicer nicht gefunden? ("Gefundene Slicer: KEINE")
--------------------------------------------------
  Schick mir den Installationspfad von Bambu/Orca (z.B.
  C:\Program Files\OrcaSlicer\orca-slicer.exe), dann passe ich die Datei an.

Beenden / Neu starten
---------------------
  Fenster schliessen beendet den Helfer (Autostart startet ihn beim naechsten
  Anmelden neu). Sofort neu starten: Datei erneut doppelklicken.
